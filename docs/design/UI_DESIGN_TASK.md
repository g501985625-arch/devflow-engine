## 设计风格

### 主色调

| 颜色 | 用途 | TailwindCSS |
|------|------|-------------|
| 深蓝 | 主色 | `blue-600`, `blue-700` |
| 青色 | 辅色 | `cyan-500`, `cyan-600` |
| 灰色 | 背景 | `gray-100`, `gray-900` |
| 白色 | 文字（深色模式） | `white` |
| 黑色 | 文字（浅色模式） | `gray-900` |

### 状态颜色

| 状态 | 颜色 | TailwindCSS |
|------|------|-------------|
| 成功 | 绿色 | `green-500` |
| 进行中 | 蓝色 | `blue-500` |
| 待执行 | 灰色 | `gray-400` |
| 错误 | 红色 | `red-500` |
| 警告 | 黄色 | `yellow-500` |

### Agent 状态

| 状态 | 颜色 | 图标 |
|------|------|------|
| 在线 | 绿色 | `●` |
| 离线 | 灰色 | `○` |
| 忙碌 | 黄色 | `◐` |

### 动效要求

- 阶段切换：平滑过渡动画（0.3s ease）
- 卡片悬停：轻微缩放 + 阴影增强
- 进度条：加载动画
- 状态切换：颜色渐变过渡
